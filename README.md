# Urban Jungle Co

Urban Jungle Co is a beautiful, modern, and simple website built using only HTML and CSS. It features a stunning store page for plants and nature-inspired interior decor products, bringing the green vibe to any space.

## 🚀 Live Demo

You can view the live website here: [Urban Jungle Co Live Preview](https://jeedasokkar.github.io/Urban-Jungle-Co/)

## 💻 Project Content & Sections

The site consists of a main page with the following sections:
- **Header:** Elegant logo and a warm welcome text
- **Features:** Our product features and what makes us unique
- **Call to Action (CTA):** Encouraging buttons to explore the store
- **Gallery:** High-quality, aesthetic plant and decor images
- **Newsletter:** A clean signup form for updates and offers

## 🛠️ Technologies Used

- **HTML5:** Semantic elements for structure and SEO
- **CSS3:** Custom styling, typography, and responsive layouts
